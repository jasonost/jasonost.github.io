CensusMapper

This was a group project spanning two courses: Professor Tapan Parikh's User Interface Design class and Professor Ray Larson's Database Management class. The interface allows a user to map a limited set of Census variables on top of a standard Google Maps interface. Users can also add various points of interest and style the layers according to their preferences. For this project, I participated in:

  • a contextual inquiry of user behavior and needs,
  • paper prototyping of the interface,
  • wireframing and low-fidelity prototyping in Balsamiq,
  • testing the interface with potential users,
  • desigining and implementing the database to back the interface, and
  • coding the site in HTML, JavaScript, and jQuery.

For more information, please take a look at the interface.

Clinical Trials Browser

This was a group project in Professor Marti Hearst's Information Visualization class. The interface is implemented entirely in D3.js, and uses data collected by the National Institutes of Health at ClinicalTrials.gov and processed by the Clinical Trials Transformation Initiative at Duke University.

The visualization primarily consists of an interactive bubble chart that allows a user to select the condition and/or intervention used in the trials in which they are interested. The surrounding line and bar charts provide a comparison between the clinical trials of interest and all clinical trials in the database. For this project, I was responsible for:

  • cleaning, processing, and analyzing data,
  • coding the D3.js visualizations and interactions, and
  • creating a web page to host the visualization.

For more information, please see the visualization.

Clinical Trials Search

This was a group project in Alex Miłowski's Information Organization Laboratory class. The interface is uses Bootstrap and Leaflet, and is built off data collected by the National Institutes of Health at ClinicalTrials.gov and processed by the Clinical Trials Transformation Initiative at Duke University.

Developing this interface required significant back-end development work, notably deduplicating trial facilities and sponsors into a set of canonical institutions, and linking those institutions with Freebase information. For this project, I was responsible for:

  • cleaning, processing, and analyzing data,
  • deduplicating facilities based on name and location, and
  • developing a list of canonical institution names and images.

For more information, please see the interface.

Interstate Migration Visualization

This was an individual project in Professor Marti Hearst's Information Visualization class. The interface is implemented entirely in D3.js, and primarily consists of a chord diagram representing the interstate migration patterns for a single year. When a user hovers over a particular state, line and bar charts appear to show historical unemployment and housing cost trends, as well as the top marginal state income tax rate, for the state of interest as well as the state(s) of greatest in- and out-migration. For this project, I was responsible for:

  • identifying, processing, and analyzing data,
  • coding the D3.js visualizations and interactions, and
  • creating a web page to host the visualization.

For more information, please see the visualization.
